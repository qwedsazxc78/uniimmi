# 快速清單（出發前/入境時/入境後）

最後更新：2026-01-20  
**標籤**：#清單 #免簽 #停留簽 #居留簽 #ARC #APRC #工作許可 #留學生 #依親 #投資

> 這份清單把常見「會被卡住」的點整理成可勾選形式；細節與例外情況請對照各主題文件。

## A. 出發前（最常卡點）

- [ ] 你是走 **免簽/過境/短期停留**，還是需要 **停留簽證（<=180 天）** 或 **居留簽證（>180 天）**？
  - BOCA（簽證規定入口）：https://www.boca.gov.tw/np-3-1.html
  - 停留簽：https://www.boca.gov.tw/np-400-1.html
  - 居留簽：https://www.boca.gov.tw/np-402-1.html
  - 免簽/落地/電子簽規定：https://www.boca.gov.tw/np-384-1.html

- [ ] 你此行目的是否會觸發「不能用免簽/訪客」的活動：**工作**、**就學**、**依親/長居**、**投資移民**？
  - 先看：`taiwan-immigration/01-entry-and-prohibited.md`

- [ ] 如果你要在台灣工作：
  - [ ] 是否確認「誰先申請、申請順序」：多數路線是 **工作許可（勞動部）→ 居留簽（BOCA）→ 入境後申請 ARC（移民署）**
  - `taiwan-immigration/03-work-and-professional.md`

- [ ] 如果你要先來找工作：
  - [ ] 是否符合 BOCA 的「尋職目的停留簽證」條件？是否有人數/名額限制、或需先行線上填表？
  - `taiwan-immigration/03-work-and-professional.md`

- [ ] 如果你要來就學：
  - [ ] 是否拿到入學文件、並符合居留簽/ARC 路線？
  - [ ] 是否了解留學生在台打工通常需申請/持有工作許可（依規定）
  - `taiwan-immigration/05-student-and-graduate.md`

- [ ] 如果你要依親（配偶/子女/父母等）：
  - [ ] 你依親對象是「台灣戶籍國民」還是「外國人在台居留者」？流程與文件通常不同。
  - `taiwan-immigration/04-family-spouse.md`

- [ ] 如果你要走永久居留（APRC）或投資移民：
  - [ ] 是否了解 APRC 常見文件：健康檢查、財產/技能、近 5 年本國與我國刑事紀錄證明等（並有部分免附例外）
  - [ ] 投資移民門檻可能涉及投資額、創造就業、或公債持有期間等
  - `taiwan-immigration/06-long-stay-aprc-naturalization.md`

- [ ] 你是否屬於 **大陸地區人民** 或 **港澳居民**？常見另適用專法與不同申請系統/限制。
  - 兩岸人民關係條例：https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=Q0010001
  - 港澳條例：https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=Q0010003

## B. 入境時（移民官可能會問/看）

- [ ] 目的明確且可提供佐證（行程、住宿、資金、回程/續程機票、入學/聘僱/依親文件等）
- [ ] 符合免簽/停留條件（例如：有效護照、足夠資金、回程/續程票等）
- [ ] 不以「訪客/免簽」身分做會被認定為違反目的/條件的事（例如：受僱、以工作為目的居留等）

## C. 入境後（不要踩雷）

- [ ] 記錄你的「停留/居留期限」與到期日；延期/延長通常有窗口期（例如：居留延長常見在到期前 3 個月內提出，依規定）
- [ ] 依你身分完成後續：
  - 工作：雇主/你是否已完成工作許可與居留登記；是否需要重入國許可等
  - 學生：是否持有效居留；打工是否先拿到工作許可
  - 依親：依親居留原因是否維持（例如依親對象長期不在境內可能影響）
