# 簽證（Visa）、VDR/eVDR、費用與保證金（Security Bond）

最後更新：2026-01-20  
**標籤**：#簽證 #免簽 #VDR #eVDR #eVISA #費用 #保證金 #SecurityBond #JIM #MYVISA

## 1) 先用官方表查「你是否需要簽證」與特殊限制

JIM 的「Keperluan Visa Mengikut Negara」頁面同時包含：

- **必須申請簽證才能入境**的國籍清單（並標註部分國籍「僅可由航空入境」）
- **VDR 相關的例外/註記**
- **ASEAN 國籍短期免簽/逾 1 個月規則的註記**（頁面有明確文字）
- **黃熱病疫苗證明（如適用）**與 KKM 連結
- **eVISA 官方入口** `https://malaysiavisa.imi.gov.my/`（並提到可用 Live Chat Support）

官方來源：  
- `https://www.imi.gov.my/index.php/perkhidmatan-utama/visa/keperluan-visa-mengikut-negara/`

### 1.1 例：頁面上「重要註記」包含哪些類型？

以該頁面「NOTA」段落為例（請以頁面最新文字為準）：

- 標註 `(*)` 的國籍「**hanya boleh masuk melalui udara sahaja**」（僅能由航空入境）
- 頁面亦提到印度國籍的**簽證豁免期限**（截至 2026-12-31 的說明）
- 對阿富汗國籍的社會訪問（Lawatan Sosial）目的，提到需先取得 VDR 核准後再辦 VTR 的註記

## 2) 什麼是 VDR / eVDR？你在什麼情境會遇到？

JIM 有「Visa Dengan Rujukan (VDR)」專頁；同時在「Keperluan Visa Mengikut Negara」亦提到：

- 對於**需要簽證才能入境**的人，若要以「讀書、工作或其他目的」入境，通常會牽涉到先取得 **VDR/eVDR 或 eVAL（核准信）**再去辦理相應簽證/入境安排（頁面有此概念描述）

官方來源：  
- `https://www.imi.gov.my/index.php/visa-dengan-rujukan-vdr/`  
- `https://www.imi.gov.my/index.php/perkhidmatan-utama/visa/keperluan-visa-mengikut-negara/`

> 實務提醒：工作/學生等長期目的，通常不是「落地給你一張 social visit pass」就完成；多數是先走核准信（VDR/eVAL）→ 才去貼簽/入境 → 再做入境後背書（endorsement）。

## 3) 簽證費用（Visa Fees）

JIM 的「Kadar Bayaran Visa」頁面提供了依國籍列示的費用表（包含 Single/Multiple Entry Visa 與 Transit Visa）。  
官方來源：`https://www.imi.gov.my/index.php/perkhidmatan-utama/visa/kadar-bayaran-visa/`

使用方式建議：

- 以該表為主，對照你的國籍找金額（單位為 RM）
- 若你是走 ESD/EMGS 等不同通道（例如工作/學生），可能還有其他線上系統費用與文件費（見 `malaysia-immigration/03-expatriate-work-passes.md` 與 `malaysia-immigration/05-student-and-graduate.md`）

## 4) Security Bond（保證金）

JIM 的「Kadar Bayaran Security Bond」頁面列出不同國籍/區域對應的保證金金額（例如 RM2000 / RM1500 / RM1000 / RM750 / RM500 / RM300 / RM200），並有「其他國家」的註記。  
官方來源：`https://www.imi.gov.my/index.php/perkhidmatan-utama/visa/kadar-bayaran-security-bond/`

> 實務提醒：security bond 往往與擔保/責任相關（例如逾期、遣返成本等風險考量）；具體適用情境、誰負擔、何時退還，建議以你實際申請的類別與官方/代理指示為準。

## 5) eVISA / 官方入口

JIM 的「Keperluan Visa Mengikut Negara」頁面指向 eVISA 官方入口：  
- `https://malaysiavisa.imi.gov.my/`
