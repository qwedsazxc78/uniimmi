# 快速清單（出發前/入境時/入境後）

最後更新：2026-01-20  
**標籤**：#清單 #入境 #簽證 #eVisa #訪客 #工作簽證 #學生簽證 #受養人 #永久居民 #NewCIES

> 這份清單把常見「會被卡住」的點整理成可勾選的形式；細節與例外情況請對照各主題文件。

## A. 出發前（最常見的卡點）

- [ ] 你是否屬於 **居港權（ROA）/入境權（RTL）/香港居民** 等「不需要一般簽證/入境許可」的身分？
  - 先搞清楚：ROA/RTL/普通居住/已定居 → `hong-kong-immigration/06-long-stay-pr-roa.md`
- [ ] 你是「訪客免簽」還是「需要訪客簽證/入境許可」？免簽可停留幾天？
  - 查表：`Visit Visa / Entry Permit Requirements` → `hong-kong-immigration/01-entry-and-prohibited.md`
- [ ] 你此行目的是否會觸發「必須先拿簽證/入境許可」：**工作、就學、創業/經商、定居、延長逗留**？
  - 原則：無 ROA/RTL 者，一般需要簽證/入境許可才能工作/就學/經商/居留或逾免簽期 → `hong-kong-immigration/01-entry-and-prohibited.md`
- [ ] 如果你要走「人才/工作」：你走哪條？是否需要先拿 offer？逗留條件是「就業限制」還是「僅限時逗留」？
  - 方案對照：`hong-kong-immigration/03-work-and-talent-schemes.md`
- [ ] 如果你要走「學生」：你是否有錄取信、能負擔學費生活費、並完成本地擔保人/監護安排（未滿 18）？
  - 學生與打工限制（NOL/兼職/實習/暑期）：`hong-kong-immigration/05-student-and-graduate.md`
- [ ] 如果你要「帶家屬」：你是否具備可被接受的關係證明與經濟/住宿支持？
  - 受養人（Dependants）：`hong-kong-immigration/04-family-dependant.md`
- [ ] 如果你要走 **New CIES**（資產投資入境）：是否已了解必須先向 **InvestHK** 做核證、以及 HK$30m Net 的淨資產與投資要求？
  - `hong-kong-immigration/03-work-and-talent-schemes.md`（New CIES 章）
- [ ] 你是否會落入「指定計畫（Specified Schemes）」：申請時要先繳 **申請費 HK$600**，獲批後另繳簽證費（HK$600/HK$1,300）？
  - `hong-kong-immigration/02-visa-and-evisa.md`

## B. 入境時（移民官可能會問/看）

- [ ] 目的明確（旅遊/商務訪客/過境/就學/工作/投資/定居）且可提供佐證（行程、住宿、邀請/聘書/入學文件、資金、回程或續程機票等）
- [ ] 不要以「訪客」身分做會被視為違反條件的事（例如：受僱、經商、就學）
  - 訪客不可做＋允許的商務活動：`hong-kong-immigration/01-entry-and-prohibited.md`

## C. 入境後（不要踩雷）

- [ ] 記錄你的 **limit of stay**（逗留期限）與條件；延長逗留常見是「到期前 4 週」提出（不同計畫可能不同）
- [ ] 你的 e-Visa（PDF 通知書）下載/列印與繳費流程是否完成？
  - `hong-kong-immigration/02-visa-and-evisa.md`
- [ ] 若為訪客：別做「就業/經商/就學」等違規活動；若屬 STV/仲裁等便利安排，要保留邀請/證明文件
- [ ] 若為就業限制（employment condition）：轉職前通常要先向入境處申請批准（部分頂尖人才通道改為通知）
  - `hong-kong-immigration/03-work-and-talent-schemes.md`
