# 澳洲永久居留（PR）與入籍（Citizenship）總覽：概念、路線、文件思維

最後更新：2026-01-20  
**標籤**：#澳洲 #永久居留 #PR #永久簽證 #入籍 #citizenship #conferral #descent #法規

> 本頁以「概念總覽」為主：PR 與入籍常被混用，但在澳洲制度上是不同狀態；請以官方指引與法規為準。

## 1) PR（Permanent residence）在澳洲制度裡代表什麼（概念）

- PR 通常是「持有永久簽證」在澳洲合法居留與工作的狀態（具體權利/旅行權與條件仍需看你持有的永久簽證與個案）。
- 官方 PR 概覽入口：`australia-immigration/sources/immi_permanent_resident_overview.md`
- 若你要對照法源（研究用）：
  - Migration Act 1958（最新文字）：`australia-immigration/sources/leg_migration_act_1958_latest.md`
  - Migration Regulations 1994（最新文字）：`australia-immigration/sources/leg_migration_regulations_1994_latest.md`

## 2) 入籍（Become an Australian citizen）：常見方式與官方入口

- 官方指出可用不同方式成為澳洲公民；並提到最常見是 **conferral** 與 **descent**：`australia-immigration/sources/immi_citizenship_become_a_citizen.md`
- 入籍研究用法源：
  - Australian Citizenship Act 2007（最新文字）：`australia-immigration/sources/leg_australian_citizenship_act_2007_latest.md`
  - Australian Citizenship Regulations 2016（最新文字）：`australia-immigration/sources/leg_australian_citizenship_regulations_2016_latest.md`

## 3) PR → 入籍的規劃方式（研究筆記）

- 先把你的「身份狀態」說清楚：目前持有哪個簽證、是否 in-effect、是否有條件限制。
  - 用 VEVO 查你目前 in-effect 簽證細節與條件：`australia-immigration/sources/immi_check_conditions_overview.md`
- 再把你要申請的路線（永居/入籍）拆成可證明要件：
  - 居住/出入境時間線（護照出入境章、行程、租約、工作/學校紀錄等）
  - 品格/健康等共同要求（依申請類別而定）：
    - 健康：`australia-immigration/sources/immi_requirements_health.md`
    - 品格：`australia-immigration/sources/immi_requirements_character.md`

## 4) 常見踩雷（概念提醒）

- 把「永居」與「公民」視為同一件事（權利義務不同，且法源不同）。
- 沒先確認「旅行權/入境權」就長期離境（部分權利可能跟旅行條件、在澳居住狀態或其他條件相關）。
- 文件準備缺乏可追溯性：沒有時間線、證據零散、無法對照每個要件。
