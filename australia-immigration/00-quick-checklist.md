# 澳洲移民/入境快速檢查清單（Quick checklist）

最後更新：2026-01-20  
**標籤**：#澳洲 #入境 #簽證 #ImmiAccount #VEVO #簽證條件 #健康檢查 #品格審查 #生物辨識 #橋接簽證 #合規

> 本頁是「你要做什麼」的勾選清單；各簽證細節請再對照 `australia-immigration/01-entry-and-visas-overview.md` 與各主題頁。

## A. 申請前（最常卡點）

- [ ] 釐清目的：觀光/探親、商務訪客、就學、工作、技術移民、家庭團聚、PR、入籍。
- [ ] 確認你要走的簽證/路線是否允許「在境內申請」或「必須在境外申請/核發」。
- [ ] 準備護照與身份文件：護照有效期、舊護照/改名紀錄（若有）、出生/婚姻/離婚/戶籍或同等證明（視簽證）。
- [ ] 建立/整理線上申請管道：
  - ImmiAccount 用途（申請簽證/公民、管理案件、上傳文件等）：`australia-immigration/sources/immi_apply_online_or_paper_online.md`
  - 注意：部分簽證（例如 ETA）不一定透過 ImmiAccount：`australia-immigration/sources/immi_apply_online_or_paper_online.md`
- [ ] 讀完你簽證類別的「Eligibility/Step guide/If you already have this visa」官方頁（用於確認最新條件與文件清單）。
- [ ] 先盤點常見共同要求（多數簽證都可能涉及）：
  - 健康（health requirement）：`australia-immigration/sources/immi_requirements_health.md`
  - 品格（character requirement）：`australia-immigration/sources/immi_requirements_character.md`
  - 英文能力（部分簽證）：`australia-immigration/sources/immi_requirements_english.md`
  - 生物辨識（部分申請人/國籍/簽證）：`australia-immigration/sources/immi_requirements_biometrics.md`

## B. 送件/審理期間（避免「以為有簽證」卻出事）

- [ ] 以官方「Visa processing times / Global processing times」評估時程與風險：
  - `australia-immigration/sources/immi_visa_processing_times.md`
  - `australia-immigration/sources/immi_global_visa_processing_times.md`
- [ ] 文件策略：先上傳「一定要有」與「最能證明核心要件」的材料，再補充。
- [ ] 如果你在澳洲境內遞交新的 substantive visa（實質簽證）申請：
  - 你可能會自動/另行取得 bridging visa（橋接簽證）來合法停留等結果；務必理解何時「in effect」與哪些條件適用。
  - BVA（010）官方說明：`australia-immigration/sources/immi_bva_010.md`
  - BVB（020）/BVC（030）/BVE（050/051）：`australia-immigration/sources/immi_bvb_020.md`、`australia-immigration/sources/immi_bvc_030.md`、`australia-immigration/sources/immi_bve_050_051.md`
- [ ] 若你需要出入境（travel）但目前在 bridging：
  - 不是所有 bridging 都支援離境後回澳；例如 BVA 明確寫到「does not support return travel」：`australia-immigration/sources/immi_bva_010.md`
  - 先以官方頁確認你需要哪一種 bridging，以及是否要先取得旅行權限。

## C. 入境前/入境後（用 VEVO 把「條件」落實成行為）

- [ ] 入境前：把你的簽證條件與有效資訊做一次「落地版摘要」：
  - VEVO 能查：哪個簽證、到期日、must not arrive after date、允許停留期間、條件（可做/不可做）：`australia-immigration/sources/immi_check_conditions_overview.md`
  - VEVO 不會顯示「尚未 in-effect 的簽證」細節（官方舉例：持 bridging 但原 substantive 未到期）：`australia-immigration/sources/immi_check_conditions_overview.md`
- [ ] 入境後：用官方「See your visa conditions」核對你這個簽證常見條件代碼與解釋：`australia-immigration/sources/immi_see_your_visa_conditions.md`
- [ ] 把你最可能踩雷的條件列出來（不同簽證差異極大）：
  - 工作限制（是否可工作、每兩週/每週時數、雇主限制、特定職業/場所限制等）
  - 就讀/出勤要求、必須持有保險（例如學生簽證通常要求 adequate health insurance/OSHC）：`australia-immigration/sources/immi_500_student.md`
  - 不得提供服務/不得直接對公眾販售（常見於訪客/商務訪客類；eVisitor 明確寫到 eVisitor 不是工作簽證、違反可能取消）：`australia-immigration/sources/immi_651_evisitor.md`

## D. 違規風險與「取消簽證」

- [ ] 把「會被取消」的風險當作規劃的一部分：訪客類違規工作、提供服務、虛假陳述/文件不實、未符合條件等，都可能導致取消或拒絕入境。
- [ ] 先熟悉官方「Cancelling a visa」入口頁（並在個案發生時再深入對照 Migration Act/Regs 與通知書）：`australia-immigration/sources/immi_cancelling_a_visa.md`
- [ ] 如果你收到取消/拒簽決定，務必確認是否有審查/上訴途徑、期限與是否可留澳（常與 bridging/條件綁在一起）。
