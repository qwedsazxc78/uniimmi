# 澳洲技術移民（189/190/491）與計分制度（SkillSelect/EOI）

最後更新：2026-01-20  
**標籤**：#澳洲 #技術移民 #SkilledMigration #SkillSelect #EOI #計分 #州擔保 #偏遠地區 #189 #190 #491

> 本頁以「研究者視角」整理：技術移民的流程邏輯、你要準備哪些類型的證明、以及用官方工具怎麼自查。

## 1) 三條常見路線：189 / 190 / 491（用官方描述抓方向）

- **189（Skilled Independent）**：給「受邀請」的技術工作者（並提到 eligible New Zealand citizens、eligible Hong Kong 或 BN(O) 護照持有人等）永久在澳洲任何地方生活工作：`australia-immigration/sources/immi_189_skilled_independent.md`
- **190（Skilled Nominated）**：被州/領地提名（nominated）的技術工作者，成為永久居民：`australia-immigration/sources/immi_190_skilled_nominated.md`
- **491（Skilled Work Regional, Provisional）**：由州/領地提名，在澳洲「區域（regional）」生活工作（屬 provisional）：`australia-immigration/sources/immi_491_skilled_work_regional.md`

> 核心差異通常落在：是否需要州/領地提名、工作/居住地是否限於 regional、以及後續銜接 PR 的設計。

## 2) SkillSelect/EOI 的基本玩法（概念）

- 技術移民常見是「先表達意向（EOI），再等待邀請（invitation）」的機制。
- 你需要準備的不是只有「分數」，而是：
  - 你的職業與技能如何被認定（含是否要技能評估/英文等，依職業與簽證而異）
  - 你的分數主張能否被文件支持（工作年限、學歷、英文成績、伴侶加分項等）
  - 若走州擔保/區域路線，是否能滿足該州/區域的額外要求（每州會不同）

## 3) 官方 Points calculator：用來做「分數假設」與缺口分析

- 官方計分工具入口：`australia-immigration/sources/immi_points_calculator.md`
- 實務用法（建議你做 3 版）：
  - **現況版**：完全以你已具備、可證明的資料填寫。
  - **可達成版**：把你短期可補齊的項目（英文成績、學歷完成、某段工作年限滿足、NAATI/社區語言等依你情況）列入。
  - **最佳情境版**：用來看天花板，但要明確標示「尚未取得」。

## 4) 文件思維：每一個分數主張都要有「可追溯證明」

- 年齡：出生日期/護照。
- 英文：官方接受的英文測驗成績與有效期（依簽證與政策）。
- 工作年限：雇主證明、薪資紀錄、稅務/社保/合約、職務說明一致性。
- 學歷：畢業證書、成績單、教育機構資料。
- 伴侶相關：關係證明、伴侶英文/技能（如適用）。
- 州提名/區域：提名信、居住/工作證明（如後續有要求）。

## 5) 常見踩雷點（策略性提醒）

- **把「算得到」當成「一定會被邀請」**：EOI/邀請有名額與競爭，且會受職業、州政策、年度設定等影響。
- **把「曾經符合」當成「現在仍符合」**：政策與細則會調整；請以官方最新資訊與你遞交當下的規則為準。
- **分數主張與文件矛盾**：例如工作期間重疊、職務內容與職業分類不一致、英文成績過期。
